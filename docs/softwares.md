# Softwares e Ferramentas

Lista oficial de softwares homologados para o time de sustentação e os respectivos caminhos para solicitação de acesso ou instalação.

!!! tip "Dica"
    Sempre verifique se você já possui a licença associada ao seu usuário no AD antes de abrir o chamado.

## Tabela de Softwares

| Nome do Software | Caminho para Abertura de Chamado (Service Desk) |
| :--- | :--- |
| **VPN GlobalProtect** | Portal de Serviços > Infraestrutura > Rede > Acesso Remoto |
| **SAP GUI 7.70** | Portal de Serviços > Sistemas ERP > Instalação Local |
| **Pacote Office 365** | Portal de Serviços > Softwares > Licenças Microsoft |
| **Jira / Confluence** | Portal de Serviços > Acessos > Ferramentas de Desenvolvimento |
| **Visual Studio Code** | *Self-Service* (Baixar via Central de Software da empresa) |
| **Docker Desktop** | Portal de Serviços > Infraestrutura > Solicitação de Admin Local |
| **DBeaver (Banco de Dados)** | Portal de Serviços > Softwares > Ferramentas de Banco |
| **Postman** | Portal de Serviços > Softwares > Desenvolvimento |

## Política de Instalação

!!! warning "Atenção"
    A instalação de softwares não homologados (Shadow IT) é estritamente proibida e passível de advertência conforme política de Segurança da Informação.

Se precisar de um software que não está na lista acima, abra um chamado de **"Nova Homologação de Software"**.
